# Popup Launcher

複数のURLをパラメータで指定し、ポップアップウィンドウで同時に開くHTMLツール

## 機能

- URLパラメータから複数のポップアップを一度に開く
- 各ウィンドウの幅・高さ・位置（左、上）をカスタマイズ可能
- デフォルト値を使用して最小限のパラメータで利用可能
- パラメータが数値でない場合は自動的にスキップ
- 全ポップアップを開いた後、ランチャーウィンドウ自身を自動的に閉じる

## パラメータ説明

各URLごとに以下のパラメータを指定できます（`n`は1, 2, 3...）：

| パラメータ | 説明 | デフォルト値 | 必須 |
|-----------|------|-----------|------|
| `URL${n}` | 開くURL | - | ✓ |
| `width${n}` | ウィンドウの幅（ピクセル） | 800 | - |
| `height${n}` | ウィンドウの高さ（ピクセル） | 600 | - |
| `left${n}` | ウィンドウの左端位置（ピクセル） | 指定なし | - |
| `top${n}` | ウィンドウの上端位置（ピクセル） | 指定なし | - |

## 使用例

### 1つのURL
```
popup_launcher.html?URL1=https://www.example.com&width1=800&height1=600
```

### 2つのURL（それぞれサイズ・位置指定）
```
popup_launcher.html?URL1=https://www.example.com&width1=800&height1=600&left1=0&top1=0&URL2=https://www.google.com&width2=600&height2=400&left2=850&top2=0
```

### 3つのURL（一部パラメータ省略）
```
popup_launcher.html?URL1=https://example.com&URL2=https://google.com&left2=100&URL3=https://github.com&width3=1024&height3=768&left3=200&top3=100
```

## 動作仕様

- URLパラメータは`URL1`から順番に読み込まれます
- URLが見つからないとループが終了します
- パラメータの値が数値でない場合は、そのURLはスキップされます
- ブラウザのセキュリティ設定により、ポップアップがブロックされる場合があります
- ユーザーのマウスクリックなど、操作の直後に実行される場合はポップアップの制限が緩くなります

## ウィンドウのオプション

開かれるポップアップには以下のオプションが設定されています：

- `toolbar=no` - ツールバー非表示
- `menubar=no` - メニューバー非表示
- `location=no` - アドレスバー非表示
- `status=no` - ステータスバー非表示
- `scrollbars=yes` - スクロールバー表示
- `resizable=yes` - リサイズ可能

## 注意事項

- 最初のURLパラメータ（URL1）を指定しないと、エラーメッセージが表示されます
- ポップアップのブロック設定がブラウザで有効な場合、ウィンドウが開かれない場合があります
- `left`と`top`パラメータは、ブラウザやOSの制限により正確に機能しない場合があります
- サードパーティのサイトをポップアップで開く場合、CORS関連のセキュリティ制限に注意してください
